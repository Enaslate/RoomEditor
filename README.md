# RoomEditor

Требования к тестовому заданию:

Проектировщик помещения
Необходимо создать редактор, состоящий из:
Рабочей области
Интерфейса, включающий в себя:
 Боковую навигационную панель, состоящую из:
Вкладки изменения размера помещения
Вкладки с изменением планировки
Боковую панель, содержимое которого зависит от вкладок, перечисленных выше
Редактор должен поддерживать следующий функционал:
Помещение:

Помещение представляет собой прямоугольный параллелепипед, у которого грани видны изнутри, а не снаружи, чтобы мы видели внутреннее пространство. 
Помещение можно приблизить, осмотреть с разных сторон.

Панель навигации:
Возможность переключаться между режимами построения и изменения размера
Режим изменения размера помещения.
На боковой панели появляются вводные поля для изменения параметров помещения – ширина, высота, длина. При нажатии кнопки сохранить на этой же панели в первый раз создается комната, в следующий раз изменяются параметры существующей комнаты.

Режим изменения планировки
На боковой панели появляются кнопка добавления окна и кнопка добавления двери. При нажатии на кнопку создается соответствующий объект, который пользователь может передвинуть на желаемую стену и поставить. При создании каждого объекта, на боковой панели появляется карточка объекта. На карточке хранится его название, размеры, кнопка удалить и кнопка изменения его параметров (ширина и длина, положение на стене)
